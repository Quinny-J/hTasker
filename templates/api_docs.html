API Documentation

Endpoints

  • Upload Script - POST /upload
    • Upload a Python script file.
    • Parameters: file (multipart/form-data)
  • Run Script - POST /run-script
    • Run a Python script immediately.
    • Parameters: script_name (string), arguments (string, optional)
  • Schedule Script - POST /schedule-script
    • Schedule a Python script to run at a specific interval or time.
    • Parameters: script_name (string), interval (int, optional), day (string, optional), time (string, optional), arguments (string, optional)
  • Cancel Job - POST /cancel-job/<job_id>
    • Cancel a scheduled job.
    • Parameters: job_id (string)
  • Get Logs - GET /logs
    • Retrieve execution logs.
  • API Documentation - GET /api/docs
    • View API documentation.
